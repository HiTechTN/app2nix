REST API

Analyze Package
POST /analyze 
curl -X POST -F "file=@package.deb" http://localhost:8000/analyze
Generate Nix Expression
POST /generate 
curl -X POST -F "file=@package.deb" http://localhost:8000/generate
Download from URL
POST /download 
curl -X POST -H "Content-Type: application/json" -d '{"url":"https://..."}' http://localhost:8000/download

CLI Usage

# Analyze package python main.py package.deb # Generate default.nix python main.py package.deb --output-dir ./mypackage # Universal analyzer python universal_analyze.py package.AppImage # Start web server python server.py # With URL python main.py --url https://example.com/package.deb
